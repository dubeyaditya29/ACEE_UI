Badge

Badges are used to convey some messages/information to the user. For example badge on a cart icon represents the quantity of items in a cart or badge on applications represents the unseen messages on the account and alot more. We have used FontAwesome for all the icons you see on the page. You can use different classes to customize the badge like badge-sm , badge-loc-top-left, badge-white to make badge looks appealing.

Badge on an icons (of variable sizes)

Add .badge-item to apply any badge to an icon or an avatar.Check the below to see examples of how to implement this in your code.
