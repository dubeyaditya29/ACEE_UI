Avatar

Avatars are images that users can set as their profile picture.These can be custom photos uploaded by users, or generated as Identicons as a placeholder.

Basic Example

Add .avatar to any element to make it an avatar.

            <img class="avatar" src="../../assets/my-img.jpg" alt="my-profile" />

Small Avatar Example

Anything less than equal 30px wide should include the .avatar-small modifier class to reset the border-radius to a more appropriate level. 

            <img class="avatar-small" src="../../assets/my-img.jpg" alt="my-profile" />

Medium Avatar Example

Anything less than equal 50px wide should include the .avatar-medium modifier class to reset the border-radius to a more appropriate level. 

            <img class="avatar-medium" src="../../assets/my-img.jpg" alt="my-profile"/>

Large Avatar Example

Anything less than equal 80px wide should include the .avatar-large modifier class to reset the border-radius to a more appropriate level. 

            <img class="avatar-large" src="../../assets/my-img.jpg" alt="my-profile" />
